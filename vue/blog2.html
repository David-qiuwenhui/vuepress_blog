    # Vue2 调试方式

    # 一、serve（npm 包）

    安装：npm install --global serve

    https://www.npmjs.com/package/serve

    使用：进入到项目构建生成的/dist目录，执行serve .

    场景：运行打包生成的前端资

    # 二、vue ui

    通过可视化的方式查看工程的插件、依赖，配置等

    还具有任务执行的仪表盘（辅助优化打包构建过程，优化项目）
